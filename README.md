# 🚗 Rahul Travels – React Project

> Developed as part of Vocational Training at **Innodeed**

---

## 📝 Project Overview

**Rahul Travels** is a modern and responsive travel booking web application built with **React**. It provides an intuitive interface for users to explore travel routes, prices, and make bookings easily.

This project was completed during my vocational training at **Innodeed**, showcasing my frontend development skills.



## 🚀 Live Demo

👉 [rahultravels-by-avinashsoni.netlify.app](https://rahultravels-by-avinashsoni.netlify.app/)



## 🔧 Tech Stack

- ⚛️ React (with Vite)
- 🎨 Tailwind CSS
- 🎞️ Framer Motion
- 🔁 React Router
